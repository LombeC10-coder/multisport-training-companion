# MultiSport Roadmap

## Phase 1 — Product Foundation

- Define project goals
- Define MVP scope
- Create GitHub repo
- Create README and roadmap
- Decide tech stack
- Create wireframes

## Phase 2 — Mobile MVP

- Set up React Native / Expo
- Build onboarding screens
- Build athlete profile setup
- Build session logging form
- Calculate training load
- Store sessions locally
- Show weekly load summary

## Phase 3 — Recovery and Safety

- Add recovery check-in
- Add pain check-in
- Add simple safety rules
- Show warning messages before intense sessions

## Phase 4 — Backend

- Add authentication
- Add database
- Save user profile
- Save sessions
- Save recovery check-ins
- Save pain records

## Phase 5 — Smarter Recommendations

- Generate weekly training insights
- Suggest lower intensity when recovery is poor
- Suggest deload when load spikes
- Add basic AI/ML later
